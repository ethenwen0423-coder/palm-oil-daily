# Vinson Research Writing Standard

本 Skill 用于统一项目内金融研究内容的写作标准，覆盖棕榈油日报、周报、A股晨报、市场点评、行业专题、网站文章、客户推送、PPT文案和 Markdown 研究报告。

目标不是“去 AI 味”，而是让研究内容接近国内头部卖方研究员的表达质量：专业、克制、逻辑严谨、观点鲜明、信息密度高，并能帮助客户做决策。

使用方式：

1. 生成研究内容前先读取 `SKILL.md`。
2. 涉及具体表达、检查、术语或反面案例时，按 `SKILL.md` 指引读取对应文件。
3. 本 Skill 只规范写作，不修改数据来源、业务逻辑或交易策略。
