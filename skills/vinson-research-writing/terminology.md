# Terminology

统一术语，避免同一报告内混用多个叫法。

## Oilseed And Futures

| Preferred | Avoid |
|---|---|
| 棕榈油 | 棕油、棕榈、P油 |
| 豆油 | 大豆油、豆油脂 |
| 菜油 | 菜籽油、菜油脂 |
| 马来西亚棕榈油（MPOB） | 首次出现不要只写 MPOB |
| 马棕 | 首次出现后可用 |
| 大商所 | DCE 可在表格中使用 |
| 郑商所 | ZCE 可在表格中使用 |
| 主力合约 | 主连、主力月混用 |
| 支撑位 | 支撑价格、下沿 |
| 压力位 | 阻力位、上沿 |
| 强弱分界 | 多空分水岭 |

## Analysis Terms

| Preferred | Avoid |
|---|---|
| 基本面 | 供需面、基本情况混用 |
| 资金面 | 资金情况 |
| 情绪面 | 市场气氛 |
| 趋势 | 走势倾向、行情方向混用 |
| 库存 | 存货、社会库存混用，除非特指 |
| 基差 | 现货贴水、升贴水混用，除非解释 |
| 价差 | spread 中英混用 |
| 比价 | ratio 中英混用 |
| 利润 | margin 中英混用 |
| 进口利润 | 进口盈亏混用 |

## Trading Terms

| Preferred | Avoid |
|---|---|
| 低吸 | 抄底 |
| 追多 | 盲目买入 |
| 止损 | 割肉 |
| 目标位 | 看到哪里 |
| 仓位 | 手数随便写 |
| 观望 | 看看再说 |
| 失效条件 | 打脸条件、错了怎么办 |
