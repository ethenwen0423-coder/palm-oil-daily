# Title Generation Examples

## 1. Mainline Unchanged

Input:

- Yesterday's Headline: `库存去化支撑棕榈油偏强震荡`
- Today's market: inventory pressure remains low, no new driver

Output:

```markdown
Headline:
库存去化继续支撑棕榈油偏强震荡

Subheadline:
马棕库存压力仍低于预期，市场继续交易供应收缩而非新增需求。

Report Title:
棕榈油日报：库存去化延续，价格维持偏强震荡

One Sentence Summary:
棕榈油短线维持偏强震荡，核心支撑仍来自库存去化延续。
```

A股短线仍是结构性轮动行情，交易重点在板块选择而非指数方向。
