# 油脂补充参考源策略

本文件中的微信公众平台链接是“样例来源池”，用于理解油脂类文章的来源类型、写作风格、产业观点维度和市场叙事方式。它们不是固定引用清单，也不是每次报告必须引用的文章。

生成日报、周报时必须优先动态搜索和核验最新同类来源，避免固定引用旧链接导致观点或数据过期。可以参考的动态来源类型包括：期货公司公众号/官网晨报周报、产业资讯公众号、油脂油料研究文章、交易所公告、官方机构发布和主流财经资讯。

不得把微信文章中的行情、库存、出口、产量、持仓、价差等数字直接包装成已核实事实；核心数据仍需优先用交易所、MPOB/MPOA/ITS/GAPKI/USDA/CME/ICE/DCE、东方财富、问财、研报搜索和期货公司研报交叉验证。

使用规则：

1. 先查找最新同类来源，再参考本文件样例链接；不得默认只引用本文件链接。
2. 优先提炼“驱动是什么、市场如何交易、分歧在哪里、风险是什么”。
3. 若文章观点与官方数据或交易所行情冲突，以官方/交易所/可核验数据为准。
4. 若引用文章观点，报告中只写概括，不长篇复制原文。
5. 若最新同类来源无法访问，在“信息来源与核验说明”中写明“微信/产业参考源访问失败或未使用”，不要影响正式报告发布。
6. 这些来源是补充参考，不替代 Reuters、Wind、官方机构、交易所和已安装金融 skills。
